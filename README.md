# 💪 Health & Macro Calculator

Estimate daily calorie needs (Mifflin-St Jeor BMR × activity), BMI with category, and turn a calorie target into a protein/carbs/fat macro split with a visual bar.

**Live demo:** https://skrizidev.github.io/health-macro-calculator/

## Pages
- **Calories** — TDEE, cut/bulk targets and BMI
- **Macros** — macro grams from calories and a chosen split
- **About** — the equations and a health disclaimer

## Tech
Vanilla HTML / CSS / JavaScript. Educational only — not medical advice.
